# Taxpayer
This project contains two Java console programs.

Deposit
-------
A simple menu-driven banking program that allows the user to:
- Show the current balance
- Clear the balance
- Deposit money (negative deposits are not allowed)
- Quit the program

The program uses loops and input validation to handle user interaction.

Proj02mrauf3
------------
A tax calculation program that works with a Taxpayer class to:
- Create taxpayers with income, age, marital status, and veteran status
- Calculate exemptions, taxable income, and tax owed
- Compare taxpayers to determine who pays the highest tax
- Collect and validate user input using Scanner

Key Concepts Used
-----------------
- Classes and objects
- Loops and conditionals
- Input validation
- Console-based user interaction
